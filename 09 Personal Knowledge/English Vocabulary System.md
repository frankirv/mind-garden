# English Vocabulary System

## Goal
Improve English vocabulary using reading, root/meaning analysis, and AI-generated examples.

## Desired Features
- Extract unknown vocabulary from documents.
- Group variants by lemma.
- Hide known words.
- Generate definitions and examples.
- Use local LLM where possible to reduce API cost.

## Word Note Template
```markdown
# word

## Meaning

## Root / Morphology

## Example Sentences

## Similar Words

## My Sentence

## Status
- [ ] New
- [ ] Learning
- [ ] Known
```
