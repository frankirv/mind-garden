# Mac Development Workstation

## Role
Main machine for coding, writing, Obsidian, architecture docs, and controlling Windows services.

## Recommended Structure
```text
~/code       -> source code
~/obsidian   -> knowledge vault
~/downloads  -> temporary files
~/scripts    -> personal automation scripts
```

## Network / Remote
- Use Parsec to control Windows.
- Use wired LAN for lower latency.
- Use same or comfortable resolution, not necessarily identical to Mac.
- Consider higher LAN speed only when current 1GbE is a real bottleneck.

## Tools
- VS Code
- GitHub CLI
- Obsidian
- Terminal / Warp / shell
- Docker client if needed
