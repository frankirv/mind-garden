# Prompt and Skill Library

## Purpose
Reusable instructions for Codex, Claude Code, and ChatGPT.

## Categories
- Architecture constraints
- Code review
- Unit test generation
- API scaffolding
- Documentation
- Refactoring
- Database migration
- UI implementation

## Suggested Repo Options
### Option A: In Each Main Repo
Best when prompts are tightly coupled to a codebase.

### Option B: Shared Prompt Repo
Best when multiple projects need common standards.

### Option C: Hybrid
Keep universal prompts in shared repo; keep project-specific constraints in the main repo.
