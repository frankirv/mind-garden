# Bible Study System

## Goal
Build a consistent Bible study system using Kindle, iPad, computer, and Obsidian.

## Device Roles
| Device | Best Use |
|---|---|
| Kindle with pen | Focused reading and light notes |
| iPad | Flexible note-taking, highlights, split screen |
| Computer | Deeper study, writing, Obsidian organization |
| Obsidian | Long-term knowledge base |

## Suggested Flow
1. Read on Kindle or iPad.
2. Capture quick notes.
3. Move key insights into Obsidian.
4. Create topic notes for theology, questions, and applications.
5. Link Bible passages to topic notes.

## Folders
- Passage Notes
- Topic Notes
- Church / Theology
- Sermon Notes
- Questions

## Weekly Practice
- [ ] Review notes from the week
- [ ] Create 1–2 permanent notes
- [ ] Link passages to themes
- [ ] Write personal application
